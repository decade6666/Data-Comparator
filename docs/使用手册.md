# 使用手册

本手册面向业务/测试/实施人员，讲解如何使用“数据集对比”GUI 工具完成跨版本 Excel 数据差异比对，并生成可视化高亮结果。

## 一、运行程序
- 方式一：双击可执行程序（若已打包）
- 方式二：命令行运行源码
  ```bash
  python run.py
  ```
- 方式三：模块方式
  ```bash
  python -m src.main
  ```

首次运行会在用户 AppData 下创建临时与配置目录，例如：
- 配置目录：`PyDataCompare/temp/configs`
- 日志与中间文件：`PyDataCompare/temp`

## 二、界面说明与流程
界面主要分为顶部操作区、进度区、左右主面板：
- 顶部操作区（开始/停止/高级设置）
- 进度区（整体进度条 + 状态文本）
- 左侧：配置管理（新建/删除/重命名/导入/导出 + 配置列表）
- 右侧：参数详情（路径选择、预设/结构、颜色、参数卡片、进阶设置）

标准流程：
1) 路径选择
   - 旧版本文件：选择旧版 Excel
   - 新版本文件：选择新版 Excel
   - 输出目录：选择结果保存位置
2) 预设与结构
   - 表头行：默认 1
   - 锚点行：默认 1
3) 颜色设置
   - 更新标记颜色（默认 `#FFE5E5`）
   - 删除标记颜色（默认 `#DC143C`）
   - 新增标记颜色（默认 `#00FF00`）
4) 参数卡片
   - 删除列（common_cols）：生成结果时统一删除的列
   - 排除表单（exclude_sheets）：比对前会移除这些 Sheet
   - 默认锚点（default_keys）：通用的行匹配锚点列
   - 指定表单锚点（sheet_key_map）：为特定 Sheet 覆盖默认锚点
5) 进阶设置
   - 最大线程数：默认 CPU 核心-1
   - 深色模式：可切换主题（部分系统不启用）
   - 合并删除数据：控制是否把“删除表单/行”也写入结果
6) 保存/加载配置
   - 左侧列表选择配置项，右上点击“保存”或使用“快速编辑/导入/导出”
7) 点击“开始”执行比对；可随时“停止”

提示：
- 文件来自互联网（受保护）会自动尝试移除 Zone.Identifier 并清理筛选器；过程会创建 `_nofilter_` 临时副本到 AppData 目录。
- 进度条从 30% 起为处理 Sheet 阶段；完成时会显示统计。

## 三、参数详解
- 旧版本文件/新版本文件：待比较 Excel 文件的完整路径
- 输出目录：结果保存位置
- 锚点行号：锚点列数据所在行的行号
- 表头行号：输出文件表头所在行的行号
- 合并删除数据：若不勾选，被删除的表单、行或列将不写入生成输出文件
- 删除列：剔除不需要对比和呈现的列
- 排除表单：完全不参与比对的 Sheet 名称
- 默认锚点：所有表单的通用锚点（锚点用于定位数据行位置，具有唯一性）
- 指定表单锚点：指定表单的锚点，优先级高于默认锚点
- 颜色：更新/新增/删除内容的标记颜色

## 四、输出结果说明
- 结果文件为 Excel：
  - 第 1 列固定为“更新情况（标记）”
  - 新增/删除表单：整表单行高亮；Sheet 标签颜色区分
  - 更新行：仅高亮发生变动的单元格
  - 列增删：表头颜色标注（新增用“新增色”，删除用“缺失色”）
- 命名与位置：存放于“输出目录”。对照日志可查看处理详情

## 五、常见问题与排错
- 文件打不开/无权限：复制后另存为再尝试；或取消 Windows “解除阻止”
- 表头与锚点识别错误：检查表头/锚点行号，避免重复
- 大文件慢/爆内存：降低“最大线程数”；关闭其它占内存程序；确认磁盘空间
- xls 格式：请转换为 .xlsx 或 .xlsm
- 输出无高亮：确认“更新情况（标记）”列存在
- 停止后残留临时文件：手动清理临时文件（AppData/temp）

## 六、快捷键与小技巧
- 配置名称过长时，鼠标悬停可见完整名称
- 列表编辑支持按行/逗号分隔批量输入
- 默认主题为浅色；若系统不支持深色主题，开关不生效
